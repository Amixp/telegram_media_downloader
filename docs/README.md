## 🔧 Технические детали

### Основные компоненты

```
telegram_media_downloader/
├── media_downloader.py      # Главный модуль
├── utils/                   # Утилиты
│   ├── config.py           # Конфигурация
│   ├── history.py          # История сообщений
│   ├── file_management.py  # Управление файлами
│   └── ...
├── docs/                    # Документация
└── tests/                   # Тесты
```

### Ключевые концепции

- **Двухуровневая архитектура истории** - JSONL для данных, HTML для отображения
- **Изоляция данных** - Уникальные ключи `(chat_id, message_id)` для каждого чата
- **Управление памятью** - Автоматическая очистка после обработки чата
- **Параллельность** - Одновременная загрузка нескольких файлов
